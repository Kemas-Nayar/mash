# 🐚 mash — A Minimal Unix Shell in C++

> *"Learn how a shell works by building one."*

`mash` is a lightweight Unix shell written in modern C++ as a systems programming project. It implements fundamental shell behavior from scratch using POSIX system calls, providing a hands-on exploration of process management, terminal control, and command-line interfaces.

---

## ✨ Features

### Built-in Commands

* `cd` — Change the current working directory
* `exit` — Exit the shell
* `cwd` — Print the current working directory
* `history` — Display command history

### External Commands

* Execute any executable available in your `PATH`
* Uses `fork()`, `execvp()`, and `wait()` to create child processes

### Interactive Line Editor

* Raw terminal mode using `termios`
* Command history
* Arrow key navigation *(work in progress)*
* Backspace support
* Prompt redrawing

### Shell Prompt

* Displays the current directory
* Shows `~` when inside the home directory
* Bash-inspired prompt style

Example:

```text
[bash@mash ~]$ ls
Desktop  Documents  Downloads

[bash@mash Documents]$ cd ..

[bash@mash ~]$ history
0 ls
1 cd ..
2 history
```

## 🎯 Motivation

This project was built to better understand how Unix shells work internally rather than relying on existing libraries such as GNU Readline. The goal is to implement core shell functionality from first principles and gain practical experience with Linux system programming.
